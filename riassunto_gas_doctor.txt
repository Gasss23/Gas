        """Costruisce il system prompt iniettando il contenuto di CLAUDE.md."""
        claude_md = root / "CLAUDE.md"
        if claude_md.exists():
            identity = claude_md.read_text(encoding="utf-8")

GAS DOCTOR IMPLEMENTATO
------------------------
- Invocazione: python gas.py doctor
- Documentato in CLAUDE.md sezione 3 (Core Commands)
- Funzione doctor() in gas.py
- Zero token LLM consumati a parte i ping minimi (max_tokens=1, ~10 token a provider)

Cosa controlla:
1. API keys: presenza GEMINI_API_KEY, GROQ_API_KEY (obbligatorie, FAIL se
   assenti) e OPENROUTER_API_KEY (opzionale, WARN se assente). Mai il valore.
2. Connettivita' provider: ping minimo per ogni brain della cascata,
   con esito OK (latenza), QUOTA (429) o KO (errore). Regge anche con
   tutti i provider giu'.
3. Integrita' file: gas.py e CLAUDE.md esistono; .gas_history.json esiste
   ed e' JSON valido.
4. Integrita' storia: zero tool result orfani; _get_window() produce una
   finestra che parte da role:user.
5. Log: dimensione gas_debug.log, WARN se supera 5MB.

Exit code: 0 se tutto OK/WARN, 1 se almeno un FAIL.
